Basic CSS Syntax

Here are some examples of basic CSS syntax:

  • Selector { Property: Value; }
  • p { color: red; }
  • #id { font-size: 24px; }
  • .class { background-color: yellow; }

Apply CSS

There are three ways to apply CSS styles:

  • External CSS file <link rel="stylesheet" href="style.css">
  • Internal styles <style> /* CSS code here */ </style>
  • Inline styles <p style="color: blue;">This text is blue.</p>

Clearfix

To clear floats, use the clearfix class:

.clearfix::after {
	content: "";
	display: table;
	clear: both;
}

Box Model

The box model consists of margin, border, padding, and content:

/* Example box model */
div {
	margin: 10px;
	border: 1px solid white;
	padding: 10px;
}
This is an example.

Selector

Selectors allow you to target specific HTML elements:

  • Element selector: p { color: green; }
  • ID selector: #my-id { font-size: 20px; }
  • Class selector: .my-class { background-color: pink; }

Property

Some common CSS properties include:

  • Background: background-color: yellow;
  • Box shadow: box-shadow: 5px 5px 5px #333;
  • Text shadow: text-shadow: 3px 3px 3px #ccc;
  • Font: font-family: Arial, sans-serif;
  • Border/outline: border: 1px solid white; outline: none;
  • Border radius: border-radius: 10px;

Media Query

Media queries allow you to apply styles based on screen size:

@media screen and (max-width: 600px) {
	body {
		background-color: pink;
	}
}
